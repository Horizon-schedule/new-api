# AI 模型 API · 引言与快速开始

## 服务简介

本 API 网关将以下模型统一在一个入口下，**一个 API Key 通用**：

| 系列 | 典型用途 | 推荐接口 |
|------|----------|----------|
| **GPT** | 通用对话、工具调用、视觉 | OpenAI 兼容 Chat |
| **通义千问 Qwen** | 中文、代码、企业场景 | OpenAI 兼容 Chat |
| **DeepSeek** | 高性价比对话与推理 | OpenAI 兼容 Chat |
| **Claude / Claude Code** | 长文本、编码、Agent | Anthropic Messages |
| **Gemini** | 多模态、Google 生态 | 原生 generateContent 或 OpenAI 兼容 |

---

## 接口地址

```
{{baseUrl}}
```

> 由服务提供商告知，例如 `https://api.example.com`。  
> **请勿** 使用 `docs.newapi.pro`——那是文档网站，不是 API 网关。

---

## 鉴权方式

所有请求需携带 HTTP 头：

```
Authorization: Bearer sk-你的API密钥
```

API Key 在控制台 **「令牌」** 中创建，请妥善保管。

---

## 快速对照：我该用哪个接口？

| 你的工具 / 场景 | 接口 | 说明 |
|-----------------|------|------|
| OpenAI SDK、Cursor、LangChain 等 | `POST /v1/chat/completions` | model 填 gpt / qwen / deepseek / gemini 等 |
| Claude Code、Anthropic SDK | `POST /v1/messages` | 需额外请求头 `anthropic-version: 2023-06-01` |
| Google Gemini SDK | `POST /v1beta/models/{model}:generateContent` | 原生 Gemini 格式 |
| 查看可用模型 | `GET /v1/models` | 返回 OpenAI 格式列表 |

---

## Claude Code 配置示例

在终端或 IDE 插件中设置：

| 环境变量 | 值 |
|----------|-----|
| `ANTHROPIC_BASE_URL` | 网关根地址（**不含** `/v1`） |
| `ANTHROPIC_API_KEY` | 控制台颁发的 `sk-xxx` |

---

## 在线调试

1. 在文档右侧 **运行** 面板选择示例（GPT / Qwen / DeepSeek / Claude / Gemini）。
2. 在 Apifox 环境中填入你的 `api_key` 后点击 **发送**。
3. 若返回 401，检查 Key 是否有效；若连接失败，检查 `baseUrl` 是否正确。

---

## 错误码说明

| HTTP 状态码 | 含义 |
|-------------|------|
| 401 | API Key 无效或未填写 |
| 403 | 无该模型权限或余额不足 |
| 429 | 请求过于频繁 |
| 502 / 503 | 上游暂时不可用，可稍后重试 |
