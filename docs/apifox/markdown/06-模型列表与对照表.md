# 模型列表与接口对照表

## 一、按厂商对照

| 厂商 | 典型 model 前缀 | HTTP 路径 | 请求格式 |
|------|-----------------|-----------|----------|
| **GPT** | `gpt-`、`o1`、`o3`、`o4` | `POST /v1/chat/completions` | OpenAI Chat |
| **Qwen** | `qwen`、`qwen2`、`qwen3` | `POST /v1/chat/completions` | OpenAI Chat |
| **DeepSeek** | `deepseek` | `POST /v1/chat/completions` | OpenAI Chat |
| **Claude** | `claude-` | `POST /v1/messages` | Anthropic Messages |
| **Claude Code** | 同 Claude | `POST /v1/messages` | `ANTHROPIC_BASE_URL` 指向网关 |
| **Gemini** | `gemini-` | `POST /v1beta/models/{model}:generateContent` 或 `/v1/chat/completions` | 原生 / OpenAI 兼容 |

---

## 二、获取完整模型列表

### OpenAI 格式（GPT / Qwen / DeepSeek / Gemini 兼容名）

```bash
curl "{{baseUrl}}/v1/models" \
  -H "Authorization: Bearer {{api_key}}"
```

响应示例：

```json
{
  "object": "list",
  "data": [
    { "id": "gpt-4o", "object": "model", "owned_by": "openai" },
    { "id": "qwen-plus", "object": "model", "owned_by": "ali" },
    { "id": "deepseek-chat", "object": "model", "owned_by": "deepseek" },
    { "id": "gemini-2.5-flash", "object": "model", "owned_by": "google" }
  ]
}
```

### Gemini 原生格式

```bash
curl "{{baseUrl}}/v1beta/models" \
  -H "Authorization: Bearer {{api_key}}"
```

## 三、客户端配置速查

| 工具 | Base URL / 环境变量 | Key |
|------|---------------------|-----|
| OpenAI SDK | `{{baseUrl}}/v1` | `sk-xxx` |
| Claude Code | `ANTHROPIC_BASE_URL={{baseUrl}}` | `ANTHROPIC_API_KEY=sk-xxx` |
| Anthropic SDK | `base_url={{baseUrl}}` | `sk-xxx` |
| Gemini REST | `{{baseUrl}}/v1beta` | Bearer `sk-xxx` |
| LangChain | `openai_api_base={{baseUrl}}/v1` | `sk-xxx` |

---

## 四、Apifox 环境变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `baseUrl` | 网关根地址 | `https://api.example.com` |
| `api_key` | 控制台令牌 | `sk-xxxxxxxx` |

文档与接口中的 `{{baseUrl}}`、`{{api_key}}` 会自动替换为环境值。

---

## 五、计费与权限

- 不同 model 单价不同，以控制台 **模型定价 / 日志** 为准。
- 若返回「无权限」或「模型不存在」，请在管理后台 **模型管理 / 渠道** 中确认该 model 已启用并对你的分组可见。
